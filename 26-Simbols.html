Nuevos tipos de datos

Comenzaremos a ver nuevos tipos de datos que se integran a partir del 2015. Para esto esto tendremos habiliatada la consola con su respectivo go live para poder ir revisando los cambios cada vez que guardamos. Un simbolo mantine su valor privado, y para uso interno. Generalmente su caracteristica es de objeto. No podemos usar el ioperador new, usaremos funcion contructora symbol, symbol nos permite crear identificadores de referencia .
