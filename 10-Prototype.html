Sobre Javascript

JS es un lenguaje multiparadigma, como la prograación funcional y la orientada objetos (POO). La más usada es la programación orientada a objetos.

Que es un Prototype

Es el mecanismo por el cual un objeto puede heredar de un objeto padre atributos y métodos (Funciones).

Ahora veremos las propiedades que tiene el POO

  • Clases:

    • Esquema o modelo a seguir para luego poder generar instancias.

  • Objetos:

    • Es una instancia o copia de una clase

  • Atributos:

    • Es una caracteristica o propiedad del objeto. (Son variables dentro de un objeto).

  • Métodos:

    • Son las acciones que un objeto puede realizar, ejemplo console.log, console.error. (Son funciones dentro de un objeto).
